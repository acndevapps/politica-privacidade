Política de Privacidade

Esta Política de Privacidade descreve como as informações são tratadas no aplicativo VolleyBall ScoreBoard.

1. Informações coletadas

O aplicativo não coleta, armazena ou compartilha informações pessoais identificáveis dos usuários.

O aplicativo funciona de forma local no dispositivo do usuário e não exige cadastro, login ou fornecimento de dados pessoais.

2. Uso de informações

Como não há coleta de dados pessoais, nenhuma informação é utilizada para fins comerciais, analíticos ou de marketing.

3. Compartilhamento de informações

O aplicativo não compartilha informações com terceiros.

4. Serviços de terceiros

Atualmente, o aplicativo não utiliza serviços de terceiros que coletem dados do usuário, como serviços de anúncios ou ferramentas de análise.

Caso futuramente sejam integrados serviços como anúncios, métricas de uso ou outras ferramentas externas, esta Política de Privacidade será atualizada.

5. Segurança

Embora o aplicativo não colete dados pessoais, são adotadas boas práticas de desenvolvimento para garantir o correto funcionamento e a segurança do aplicativo.

6. Alterações nesta política

Esta Política de Privacidade pode ser atualizada a qualquer momento. Recomenda-se que o usuário revise periodicamente esta página para estar ciente de eventuais alterações.
